## 💡 登录规则说明

### 服务器验证逻辑
```java
// 位置：xiangqi-server/.../GameServer.java
boolean authenticated = 
    username != null && !username.trim().isEmpty() && 
    password != null && !password.trim().isEmpty();
```

### 规则解释：
✅ **接受条件**：
- 用户名不为 null
- 用户名去除空格后不为空
- 密码不为 null
- 密码去除空格后不为空

❌ **拒绝条件**：
- 用户名为空或只有空格
- 密码为空或只有空格

2. 启动服务器：
   ```bash
   start-server.bat
   ```

3. 启动客户端：
   ```bash
   start-client.bat
   ```

4. 在登录界面输入任意非空用户名和密码，点击登录。
