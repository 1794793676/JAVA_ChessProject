# 登录说明 / Login Guide

## 📝 登录信息

### ⭐ 重要提示
本游戏**不需要注册**，可以直接登录！

### 🔑 默认登录方式

**用户名和密码可以是任意值，但不能为空！**

系统会接受任何非空的用户名和密码组合。

### ✅ 有效的登录示例

| 用户名 | 密码 | 说明 |
|--------|------|------|
| player1 | 123456 | ✅ 有效 |
| 小明 | password | ✅ 有效 |
| test | test | ✅ 有效 |
| user | abc | ✅ 有效 |
| 张三 | 任意密码 | ✅ 有效 |

### ❌ 无效的登录示例

| 用户名 | 密码 | 说明 |
|--------|------|------|
| (空白) | 123456 | ❌ 用户名为空 |
| player1 | (空白) | ❌ 密码为空 |
| (空白) | (空白) | ❌ 都为空 |

## 🎮 使用步骤

### 1️⃣ 启动服务器
首先运行：
```bash
start-server.bat
```
等待看到 "Game server started on port 8888" 提示。

### 2️⃣ 启动客户端
然后运行：
```bash
start-client.bat
```

### 3️⃣ 登录
在登录界面：
1. 输入任意用户名（不能为空）
2. 输入任意密码（不能为空）
3. 点击"登录"按钮或按回车键

### 4️⃣ 进入游戏
登录成功后，将进入游戏大厅，可以创建房间或加入其他玩家的房间。

## ⚠️ 常见问题

### Q: 为什么提示 "Message validation failed"？
**A:** 这通常是因为：
- 用户名或密码为空
- 没有先启动服务器
- 服务器地址或端口配置错误

**解决方法**：
1. 确保输入框不为空
2. 确认服务器已启动
3. 检查 `client.properties` 中的服务器配置

### Q: 可以使用中文用户名吗？
**A:** 可以！支持任何 Unicode 字符作为用户名。

### Q: 密码会被保存吗？
**A:** 不会。这是一个演示版本，不会保存任何用户信息。

### Q: 多个客户端可以使用相同的用户名吗？
**A:** 可以，但可能会造成混淆。建议使用不同的用户名。

## 📋 配置文件

如需修改服务器地址或端口，编辑 `client.properties`：

```properties
# 服务器地址
client.serverHost=localhost

# 服务器端口
client.serverPort=8888
```
